## 🌠背景介绍
**「使徒来袭！」**

**「勇敢的少年啊快去创造奇迹！」**

## 🖥环境要求
1. python 3.x+
2. pygame

## 🕹操作说明
* 发射：空格
* 飞行操作：←左键向左移动，→右键向右移动

## ⚔开始战斗
1. clone我的仓库
```
$ git clone https://github.com/HauUhang/Alien_Invasion.git

$ cd Alien-Invasion
```
2. 运行
```
$ python alien_invasion.py
```
